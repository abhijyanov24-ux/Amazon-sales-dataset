# Amazon Sales Analysis

##  Project Overview
This project focuses on analyzing Amazon sales data to uncover meaningful insights such as sales trends, top-performing products, and regional performance.

##  Objectives
- Analyze total sales and profit
- Identify top-selling products and categories
- Understand city-wise sales performance
- Track monthly sales trends

##  Tools & Technologies Used
- Python (Pandas, NumPy)
- SQL
- Power BI

##  Key Insights
- Electronics category generated the highest revenue
- Major sales came from top cities
- Sales peaked during specific months

##  Project Structure
- Dataset (CSV file)
- Python analysis script
- SQL queries
- Power BI dashboard
- README file
